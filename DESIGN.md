# 设计文档

## 视觉风格

赛博朋克 + 科技感

### 颜色方案
- 主色调：青蓝色 (#00f5ff)
- 辅助色：紫红色 (#ff00ff)
- 背景色：深色 (#0a0a0f)
- 强调色：黄色 (#ffff00)

### 字体
- 标题：Orbitron（那种科技感的等宽字体）
- 正文：Rajdhani（现代感的无衬线字体）

### 动画效果
- 网格背景移动
- 粒子浮动
- 按钮发光悬停
- 连线动态效果

## 界面布局

```
┌─────────────────────────────────────────────────────┐
│  顶部状态栏：Logo + 系统状态                          │
├──────────┬─────────────────────────────┬───────────┤

```

## 组件设计

### Agent Card（智能体卡片）
```
┌─────────────────┐
│  📋            │  <- 图标
│  规划智能体    │  <- 名字
│  任务规划      │  <- 角色描述
└─────────────────┘
```

### Workflow Node（工作流节点）
```
        [输入]
┌─────────────────────┐
│  📋 规划智能体    │
│  [状态: idle]       │
│              [输出] │
└─────────────────────┘
```

### Connection Line（连线）
- 贝塞尔曲线
- 发光效果
- 虚线表示临时连线

## 交互流程

### 1. 拖拽节点
```
鼠标按下节点 -> 记录偏移 -> 拖动 -> 更新位置 -> 重绘连线
```

### 2. 连线
```
点击输出端口 -> 画临时线 -> 鼠标移动 -> 点击输入端口 -> 保存连线
```

### 3. 执行
```
点击执行 -> 从入口节点开始 -> 执行动画 -> 更新日志 -> 完成
```
