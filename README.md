# Hermes AI Orchestrator - 赫尔墨斯AI编排器

一个可视化的 AI 工作流编排工具，可以拖拽节点构建工作流，赛博朋克风格的界面。

## 如何使用

最简单的方式：直接打开 `standalone.html`，就可以用了。

其他功能：
- 从左边拖拽 Agent 到画布
- 点击节点的输出端口连接到另一个节点的输入
- 点执行按钮看效果
- 有几个快速模板可以直接用

## Agent 类型

- 📋 规划智能体 - 任务规划
- 🔍 研究智能体 - 信息收集
- 💻 编码智能体 - 代码生成
- ✅ 审核智能体 - 质量检查
- ✍️ 写作智能体 - 内容创作
- 🎨 设计智能体 - UI设计
- 🧪 测试智能体 - 功能测试
- 📊 分析智能体 - 数据分析

## 界面说明

- 左边：Agent 列表和快速模板
- 中间：画布和工具栏
- 右边：日志、指标、配置

## 技术栈

- 前端：原生 JS + CSS
- 后端（概念）：Express + Node.js
- 设计：赛博朋克风格

## 项目结构

```
hermes-orchestrator/
├── standalone.html      # 单文件版本，直接用
├── client/              # 前端代码
├── server/              # 后端代码（概念）
├── docs/                # 文档
└── tests/               # 测试
```
