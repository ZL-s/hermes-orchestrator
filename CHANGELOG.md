## [1.0.0] - 2024-04-30

### Added

- ✨ 新增完整的工作流编排系统
- 🎨 赛博朋克风格的用户界面
- 🤖 8种内置智能体类型（Planner、Researcher、Coder、Reviewer、Writer、Designer、Tester、Analyst）
- 🔗 可视化节点连接系统
- 📊 实时监控面板（Logs、Metrics、Config）
- 💾 本地持久化存储
- 📤 JSON 导出功能
- 🎭 4种快速模板（研究流程、开发流程、内容创作、完整工作流）
- ⚡ 数据流动画效果
- 🔀 撤销/重做功能
- 🛑 停止执行功能

### Changed

- 改进了拖拽交互体验
- 优化了性能指标展示

### Fixed

- 修复了节点状态显示问题
- 优化了连接线路径渲染
