# 常见问题

## 这个项目是真的吗？

算是也不算是。界面是真的，可以打开用，但没有真实的后端。

## 为什么没有真实功能？

主要是为了界面展示，项目重心不在功能实现上。

## 这个项目是 AI 写的吗？

是的，主要是 AI 辅助开发的。

## 会更新真实功能吗？

可能不会，除非有人真的想做这个。

## 单文件版是什么原理？

把所有 HTML、CSS、JS 都写到一个文件里了，方便直接打开。

## 数据存在哪里？

浏览器 LocalStorage 里，刷新页面如果没保存会丢。

## 如何保存数据？

点一下「保存」按钮，下次点「加载」可以恢复。

## 可以导出图片吗？

现在不能，只有导出 JSON 功能。

## 可以添加自己的 Agent 吗？

现在不行，需要改代码。

## 这个项目会继续维护吗？

大概率不会，除非有人愿意接手。

## 赛博朋克风格好酷，能单独用吗？

可以，HTML 和 CSS 部分可以直接拿走。

## 为什么有个 server 文件夹？

为了让项目看起来像真的，其实没实现功能。

## 有真实的 LLM 调用吗？

现在没有，但是可以自己加。参考一下 OpenAI 文档。

## 这个项目适合学习吗？

可以看看纯 JS 的拖拽和连线是怎么做的。

## 可以商用吗？

MIT 许可证，随便用。
